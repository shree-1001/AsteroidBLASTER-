# 🚀 Asteroid BLASTER 🛡️💥  
Responsive Space Shooter Game with Firebase Leaderboard

---

## 🌌 Overview  
Asteroid BLASTER is a fast-paced browser-based space shooter game built using HTML, CSS, and JavaScript. Control your spaceship, destroy asteroids, defeat enemy ships, and survive as long as possible while climbing the global leaderboard.

---

## 🎮 Features  

- 🎯 Smooth & responsive gameplay  
- 🚀 Spaceship movement with keyboard controls  
- 🎯 Mouse-based aiming system  
- 🔫 Shooting mechanics with cooldown  
- ☄️ Asteroids with dynamic difficulty  
- 👾 Enemy ships with AI attacks  
- ⚡ Power-ups system (Shield, Rapid Fire, Triple Shot)  
- ❤️ Health system with refill mechanics  
- 💥 Particle explosion effects  
- 📈 Level progression system  
- 🌐 Firebase-powered global leaderboard  

---

## 🧠 Tech Stack  

- **Frontend:** HTML5, CSS3, JavaScript  
- **Rendering:** Canvas API  
- **Backend:** Firebase  
  - Authentication (Anonymous Login)  
  - Firestore Database
